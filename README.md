# Putty-Zeroes-Mod
Putty Zeroes Mod from modern version putty

Zeroes Mod Putty v0.84 

Features:
Far2l extension
OSC52 support [dynamic buffer always ON]
Auto-login password
Change font size with CTRL[Shift]+MOUSEWHEEL
Fix Modern Standby

better use together with WinSSHTerm!
https://winsshterm.blogspot.com
